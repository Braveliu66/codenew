# 前端页面规格

## 1. 首页 / 新建项目页

首页就是新建项目入口，不做营销落地页。

页面顶部显示：

- 系统名称。
- CPU 使用率。
- GPU 使用率。
- 显存占用。
- 当前训练中项目。

训练中项目展示规则：

- 只有 1 个训练项目时，直接显示项目名称、进度条、当前阶段和预计剩余时间。
- 多个训练项目时，默认折叠为列表摘要，展开后显示每个项目的名称、阶段、进度和剩余时间。

主体提供两种创建方式：

| 入口 | 说明 |
| --- | --- |
| 上传图片或视频 | 进入完整素材上传流程 |
| 实时视频 | 进入实时摄像头重建流程 |

## 2. 完整素材上传页

页面目标：用户上传图片或视频，先做极速预览，再决定补传素材或精细重建。

核心区域：

- 项目名称输入。
- 标签编辑。
- 图片或视频上传区。
- 已上传素材列表。
- 素材统计面板。
- 极速预览按钮。
- 预览 Viewer。
- 精细重建按钮。

上传素材能力：

- 支持图片多选上传。
- 支持视频上传。
- 支持继续补传。
- 支持删除已上传素材。
- 图片显示缩略图。
- 点击缩略图查看大图。
- 视频显示封面帧、大小、时长、分辨率。

素材统计：

- 图片数量。
- 视频数量。
- 总文件大小。
- 平均分辨率。
- 检测到的模糊图片数量。
- 视角覆盖情况。

极速预览运行时展示：

- 当前阶段。
- 进度条。
- 预计剩余时间。
- Worker 或 GPU 信息。
- 错误信息。

极速预览完成后展示：

- 粗糙 3DGS 重建模型。
- 帧率。
- 当前 LOD。
- 重建质量提示。
- 补拍建议，例如缺少背面、侧面、顶部视角，或部分图片模糊。

用户决策：

- 如果不满意，可以继续补传或删除素材后重新极速预览。
- 如果满意，可以点击精细重建。

精细重建运行时展示：

- 当前阶段。
- 进度条。
- 预计剩余时间。
- 日志摘要。
- 失败原因。

## 3. 实时视频页

页面目标：用户通过摄像头实时采集，旁边同步显示粗重建效果。

布局：

- 左侧：摄像头画面。
- 右侧：实时 3DGS 粗重建 Viewer。
- 底部或侧边：录制控制和状态信息。

录制期间显示：

- 录制时长。
- 当前帧率。
- 已采集帧数。
- 估计视频大小。
- 实时重建状态。
- GPU 或 Worker 状态。

操作：

- 开始录制。
- 暂停或结束录制。
- 结束后重拍。
- 结束后进入精细重建。

结束录制后展示：

- 视频大小。
- 录制时长。
- 帧数。
- 分辨率。
- 粗重建效果。
- 是否建议补拍或重拍。

## 4. 项目管理页

页面目标：用户查看所有项目，并快速进入详情、删除、导出或分享。

项目以卡片显示：

- 重建效果图。
- 项目名称。
- 创建或更新时间。
- 标签。
- 输入类型。
- 项目状态。
- 文件大小。
- 若训练中，显示进度、阶段、训练时间和剩余时间。

训练中项目预览图规则：

- 若已有预览模型，使用预览截图。
- 若没有预览模型，使用上传图片或视频中的某一帧。

筛选和搜索：

- 按名称搜索。
- 按标签筛选。
- 按时间筛选。
- 按状态筛选：训练中、已完成、失败、已取消。

卡片操作：

- 查看详情。
- 删除。
- 导出 PLY。
- 分享。

## 5. 项目详情页

页面目标：查看最终模型、项目元信息、训练素材和导出分享入口。

主要区域：

- 3DGS Viewer。
- 项目信息面板。
- 操作区。
- 训练素材折叠区。
- 任务历史和日志摘要。

Viewer 要求：

- 支持全屏。
- 优先保证帧率。
- 帧率稳定后自动提高清晰度。
- 支持查看预览模型和精细模型。
- 若精细重建未完成，显示预览模型或训练中状态。

项目信息：

- 名称。
- 标签。
- 创建时间。
- 更新时间。
- 输入类型。
- 总大小。
- 当前状态。
- 模型格式和产物大小。

操作：

- 精细重建。
- 导出 PLY。
- 可选导出 OBJ / GLB。
- 分享。
- 删除。

训练素材：

- 默认折叠，不占用主要界面。
- 展开后显示图片缩略图或视频信息。
- 支持查看大图。

## 6. 用户总览页

用户可以查看：

- 项目总数。
- 训练中项目数。
- 已完成项目数。
- 失败项目数。
- 总存储占用。
- 最近任务。
- 最近反馈状态。

## 7. 管理员页面

管理员可以查看：

- 系统 CPU、GPU、显存占用。
- 每张 GPU 的任务占用。
- Worker 心跳。
- 队列长度。
- 所有用户项目数。
- 每个用户存储占用。
- 每个用户训练占用。
- 失败任务和错误日志。
- 算法许可证登记。

## 8. 问题反馈页

用户可以提交：

- 问题标题。
- 问题描述。
- 关联项目。
- 截图或附件。
- 联系方式。

用户可以查看反馈处理状态。

## 9. 当前实现同步

- 已新增 `/login`，支持用户名/密码登录和注册，token 存储在浏览器本地并随 API 请求自动附带 Authorization header。
- 未登录访问工作台页面会跳转登录页；`/about` 作为公开合规页面保留可访问。
- `/admin` 会先读取当前用户，非管理员显示无权限状态，管理员才加载任务、worker、资源和算法登记。
- 项目详情页只有后端返回真实 `preview_spz` artifact 时才加载 Spark Viewer；无产物或失败时显示 unavailable/error。
- 实时视频、精细重建和 Mesh 导出仍显示未接入，不伪装为可用能力。
